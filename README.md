# FableModdingTools
Regroup modding tools to make mods for **Fable - The Lost Chapters** and **Fable Anniversary** easier.

## Purpose of this project
This project regroups the following tools in the same place:
* [Chocolate Box](https://www.nexusmods.com/fablethelostchapters/mods/32)
* [Fable Explorer ShadowNet](https://www.nexusmods.com/fablethelostchapters/mods/3)
* [Freeroam](https://www.nexusmods.com/fableanniversary/mods/25)

These tools can be used to mod ``Fable - The Lost Chapters`` (FTLC) and ``Fable Anniversary`` (FA).

The FA-SYMLINKER.bat script allows to quickly:
* Make a backup of FTLC
* Enable all permissions to mod FTLC and FA
* Make a symlink between FTLC and FA to launch the tools the same way for both games

## Tools Setup & Use
1) Go to the [Release](https://github.com/lgbarrere/FableModdingTools/releases) section and download the last available FA-SYMLINKER.zip
2) Unzip it to a desired location
3) Find and run the FA-SYMLINKER.bat script and follow the instructions
